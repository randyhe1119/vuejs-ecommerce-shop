# internet-shop

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and shows errors and warnings
```
npm run lint
```

### Fix js errors 

``` node_modules/.bin/eslint ./server --fix```


### MongoDB start
```sudo service mongod start```
